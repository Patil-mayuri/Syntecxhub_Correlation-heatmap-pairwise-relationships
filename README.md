# Heart Disease EDA & Correlation Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Heart Disease dataset to understand the relationship between medical attributes and the presence of heart disease.

The main objective is to analyze patterns, visualize important features, and identify correlations between variables using Python.

---

## 📊 Dataset
Dataset used in this project: UCI Heart Disease Dataset

The dataset contains medical attributes such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Maximum Heart Rate
- Exercise Induced Angina
- Oldpeak
- Target (Heart Disease Presence)

Target Variable:
- 0 → No Heart Disease
- 1 → Heart Disease Present

---

## 🔍 Key Insights

Some important insights discovered from the analysis:

- Chest pain type shows strong correlation with heart disease.
- Maximum heart rate is positively related to heart disease presence.
- Oldpeak has a negative correlation with heart disease.
- Age and cholesterol have moderate influence.

These patterns help in understanding which clinical features are important indicators of heart disease.
